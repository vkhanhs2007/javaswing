📋 TÓM TẮT DỰ ÁN

Đây là một hệ thống quản lí quán cafe hoàn chỉnh được xây dựng bằng:
- Java Swing: Giao diện người dùng
- SQL Server: Lưu trữ dữ liệu
- JDBC: Kết nối database
- Tư pháp OOP: Code chuyên nghiệp

🎯 CHỨC NĂNG CHÍNH

1️⃣  QUẢN LÍ BÀN
   ✓ Xem danh sách bàn (JTable)
   ✓ Thêm bàn mới
   ✓ Xóa bàn
   ✓ Xem trạng thái (Trống/Đang phục vụ)
   ✓ Right-click menu để thêm/xóa nhanh

2️⃣  QUẢN LÍ THỰC ĐƠN
   ✓ Xem danh sách 15+ món
   ✓ Thêm món mới (tên, giá, loại, mô tả)
   ✓ Sửa thông tin món
   ✓ Xóa món (ẩn đi)
   ✓ Lọc theo loại (Cà Phê, Nước, Thức ăn)
   ✓ Giảm giá theo khung giờ

3️⃣  GỌI MÓN & HÓA ĐƠN
   ✓ Chọn bàn → Gọi món
   ✓ Chọn loại mon từ dropdown
   ✓ Click món → Nhập SL → Thêm vào hóa đơn
   ✓ Xem tổng tiền tự động
   ✓ Xem điểm tích lũy (1000đ = 1 điểm)
   ✓ Lưu hóa đơn tạm (chưa thanh toán)
   ✓ Xóa món khỏi hóa đơn

4️⃣  THANH TOÁN
   ✓ Xem tổng tiền
   ✓ Áp dụng giảm giá %
   ✓ Nhập thông tin khách (SĐT, tên)
   ✓ Dùng điểm tích lũy để trừ giá
   ✓ In hóa đơn (console)
   ✓ Cập nhật trạng thái bàn → Trống

5️⃣  QUẢN LÍ HÓA ĐƠN
   ✓ Xem danh sách hóa đơn
   ✓ Lọc theo thời gian (Ngày/Tháng/Quý/Năm)
   ✓ Xem chi tiết từng hóa đơn
   ✓ Thống kê doanh thu

6️⃣  THỐNG KÊ
   ✓ Top 10 món được gọi nhiều nhất
   ✓ Top 10 món được gọi ít nhất
   ✓ Tổng doanh thu theo thời gian

7️⃣  ĐỀ MẬT & PHÂN QUYỀN
   ✓ Đăng nhập người dùng
   ✓ 2 vai trò: Admin & Nhân viên
   ✓ Admin: Toàn quyền + Quản lí người dùng
   ✓ Nhân viên: Chỉ gọi món & thanh toán
   ✓ Đăng xuất

8️⃣  TÍNH NĂNG BỔ SUNG
   ✓ Mã tích lũy (cứ 1000đ = 1 điểm)
   ✓ Sử dụng điểm để mua hàng
   ✓ Giảm giá theo khung giờ
   ✓ Chuyển bàn khi gọi món
   ✓ Right-click để menu nhanh
   ✓ Thống kê chi tiết

🗄️ SCHEMA DATABASE

[Tables] - Bàn ăn
├── TableID (Primary Key)
├── TableName
├── Capacity
└── Status (Trống/Đang phục vụ)

[MenuItems] - Thực đơn
├── MenuItemID (Primary Key)
├── ItemName
├── Category
├── Price
├── Description
├── Available
├── Discount (%)
├── TimeDiscountStart
└── TimeDiscountEnd

[Customers] - Khách hàng
├── CustomerID (Primary Key)
├── PhoneNumber (Unique)
├── CustomerName
└── Points

[Invoices] - Hóa đơn
├── InvoiceID (Primary Key)
├── TableID (Foreign Key)
├── CreatedDate
├── PaidDate
├── TotalAmount
├── DiscountAmount
├── PointUsed
└── Status (Tạm/Đã thanh toán)

[InvoiceDetails] - Chi tiết hóa đơn
├── DetailID (Primary Key)
├── InvoiceID (Foreign Key)
├── MenuItemID (Foreign Key)
├── Quantity
├── UnitPrice
├── Discount
└── Amount

[Users] - Người dùng
├── UserID (Primary Key)
├── Username (Unique)
├── Password
├── Role (Admin/Nhân viên)
├── FullName
└── Active

⚡ CÁCH TÁI NHANH NHẤT

1. Cài SQL Server 2019 Express + JDK 11+
2. Tải JDBC driver vào lib/
3. Chạy database_setup.sql
4. Double-click run.bat (Windows) hoặc ./run.sh (Linux)
5. Đăng nhập: admin / 123456789
6. Bắt đầu sử dụng!

Chi tiết xem: QUICK_START.txt
