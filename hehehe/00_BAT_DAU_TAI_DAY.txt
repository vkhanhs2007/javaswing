❌ TROUBLESHOOTING


Lỗi: "Cannot find SQL Server"
→ Kiểm tra SQL Server chạy: Services > SQL Server status = Running
→ Kiểm tra port: netstat -an | findstr :1433

Lỗi: "Login failed for user sa"
→ Kiểm tra password: 123456789
→ Kiểm tra SQL Server: Mixed Authentication mode

Lỗi: "No suitable driver found"
→ Kiểm tra JDBC jar trong lib/ folder
→ Tên file phải có: mssql-jdbc-*.jre11.jar

Lỗi: "Failed to compile"
→ Kiểm tra Java version: javac -version (>= 11)
→ Kiểm tra path: set JAVA_HOME=C:\Program Files\Java\jdkXX

Lỗi: "Database already exists"
→ Xóa database CafeManager rồi chạy lại script
→ Hoặc: SSMS > Databases > Right-click > Delete
